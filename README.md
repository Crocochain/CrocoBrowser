## Security and Privacy ğŸ”’

Croco Browser safeguards you in real-time against:

- Intrusive ads: banners, pop-ups, video ads, and display ads  
- Pop-ups, cookies, and unwanted redirects: annoying windows, third-party cookies, and forced navigation  
- Tracking and online surveillance: ad trackers, invisible pixels, and script-based monitoring  
- Phishing and online fraud: fake websites imitating trusted services, credential theft attempts  
- Cryptoscams: fraudulent crypto websites or fake investment schemes  
- Malicious downloads: infected files, hidden malware, and unwanted software  
- Malicious domains: unsafe websites spreading viruses or scams  
- Malicious scripts and resources: JavaScript, iframes, or injected content attempting exploits  

All security features run continuously, with user data stored locally and no personal information collected.
